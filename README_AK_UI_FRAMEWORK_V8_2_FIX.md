# AK UI Framework v8.2 — Core

Este parche añade la librería visual compartida de AK Cloud dentro de Autokeys Core y corrige el error:

`Cannot find module './AKSidebar'`

## Instalación

Copia el contenido del ZIP encima del repositorio `autokeys-lab`, confirma la sustitución, haz commit y push.

No requiere SQL.

## Componentes incluidos

- AKPageShell
- AKSidebar
- AKTopbar
- AKApprovalGate
- AKCard
- AKButton
- AKInput
- AKBadge
- AKStatCard
- AKEmptyState
- AKLoader
