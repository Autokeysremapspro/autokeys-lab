# AK Core UI Refresh v2

Actualización visual sin modificar la lógica de negocio ni las opciones existentes.

## Mejoras incluidas
- Nuevo AppShell premium y responsive.
- Sidebar compacto/expandible con navegación jerarquizada.
- Topbar moderna con contexto de página, búsqueda y notificaciones.
- Diseño global renovado: fondos, tarjetas, tablas, formularios, botones y estados.
- KPI y acciones rápidas rediseñados.
- Mejor experiencia móvil y menú lateral accesible.
- Microanimaciones y accesibilidad para movimiento reducido.
- Fix anterior de Supabase/Vercel incluido.

## Instalación
Sustituir el contenido del repositorio por esta versión, conservar las variables de entorno de Vercel y desplegar con limpieza de caché.

## Verificación
Build de producción completado con Next.js 14.2.23 usando variables de entorno de prueba válidas.
