# AK Cloud Admin v7

Este sprint convierte Autokeys Core en el centro de control de AK Cloud.

## Pasos

1. Ejecuta en Supabase:

```text
supabase/autokeys_core_akcloud_admin_v7.sql
```

2. Copia el contenido del ZIP encima del repo `autokeys-lab`.
3. Commit + push.
4. Abre en Core:

```text
/ak-cloud/admin
```

## Incluye

- Gestión de soluciones / servicios del portal.
- Precios y créditos por servicio.
- Planes de suscripción editables.
- Métodos de pago: PayPal, Bizum, transferencia.
- Branding y textos del portal.
- Menú lateral actualizado con AK Cloud Admin.

## Nota

AK Cloud deberá leer estas tablas para que los cambios se reflejen automáticamente en el portal.
