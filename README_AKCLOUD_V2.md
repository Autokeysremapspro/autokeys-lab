# AK Cloud V2 — instalación

Incluye:
- precio final editable desde Autokeys Core;
- múltiples versiones V1, V2, V3 por pedido;
- selección de versión final;
- cierre exclusivamente manual y reapertura;
- formulario de nuevo pedido mejor estructurado;
- aceptación legal obligatoria y registrada.

## Paso obligatorio
Ejecuta en Supabase SQL Editor el archivo:
`supabase/migrations/20260717_akcloud_v2_versiones_precio_legal.sql`

Después despliega primero Core y después el portal. Ambos deben usar el mismo proyecto Supabase.
